# AstroHopper KR

망원경 스타 호핑을 위한 한국어 웹 앱 — [원본 AstroHopper](https://artyom-beilis.github.io/astrohopper.html) 기반 개선판

**▶ 바로 사용하기:** https://bih-lab.github.io/Astrohopper_BIH.ver/

---

## 기본 사용법

1. 스마트폰 브라우저에서 위 링크 접속
2. 위치 정보 및 기기 센서 권한 허용
3. 망원경에 스마트폰 고정 후 **정렬** 버튼으로 기준별 설정
4. **가이드 ▲** 버튼으로 주변 천체 목록 열람
5. 목록에서 천체 탭 → 자동으로 지도로 복귀 + 방향선 표시

---

## 추가된 기능

### 1. 한국어 UI

원본의 영어 인터페이스 전체를 한국어로 번역하였습니다.

- 버튼, 메뉴, 안내 메시지 모두 한국어
- 언어 선택 드롭다운에서 한국어/영어 전환 가능
- 시작 가이드 팝업 한국어화

---

### 2. 관측 가이드 패널

화면 우하단의 **가이드 ▲** 버튼을 탭하면 현재 망원경이 향하는 방향 주변의 천체 목록을 표시합니다. 스타 호핑 중 다음 목표 천체를 빠르게 찾을 수 있습니다.

#### 화면 구성

```
[지도 화면]                         [가이드 패널]
┌─────────────────────┐             ┌──────────────────────────┐
│                     │  가이드 ▲   │ [← 지도로]  RA 5h32m  +22.1°│
│   별자리 지도        │ ──────────▶ │ 반경[━●━]10°  등급[━━●]9.0  │
│                     │             │ [산개][구상][성운][은하][이중]│
│            [가이드▲]│             ├──────────────────────────┤
└─────────────────────┘             │ 천체     종류   등급  거리  │
                                    │ ● M1     성운  ★8.4  2.3°  │
                                    │ ● NGC1981 산개 ★4.6  3.1°  │
                                    │ ● M42    성운  ★4.0  5.8°  │
                                    │ ...                        │
                                    └──────────────────────────┘
```

- 천체 탭 → 패널 닫힘 + 해당 천체로 방향선 표시 + 상단 검색창에 이름 표시
- 패널 바깥 영역 탭 → 패널 닫힘

#### 필터 및 정렬

| 항목 | 범위 / 옵션 | 기본값 |
|------|------------|--------|
| 반경 슬라이더 | 5° ~ 30° | 10° |
| 한계등급 슬라이더 | 4.0 ~ 14.0 mag | 9.0 |
| 천체 종류 토글 | 산개성단 / 구상성단 / 성운 / 은하 / 이중성 | 이중성 제외 활성 |
| 정렬 방식 | 거리순 / 밝기순 | 거리순 |

#### 목록 표시 정보

| 컬럼 | 내용 |
|------|------|
| ● | 천체 기호 (고도 < 20°이면 흐리게 표시) |
| 천체 | 이름 (M1, NGC1981 등) |
| 종류 | 산개성단 / 구상성단 / 성운 / 은하 / 이중성 |
| 등급 | 겉보기 등급 (★ 기호) |
| 거리 | 현재 조준 방향과의 각거리 (°) |

> **흐린 글씨**: 현재 고도 20° 미만 — 지평선 근처라 관측 조건이 나쁜 천체

#### 반응형 레이아웃

| 화면 크기 | 패널 위치 |
|----------|----------|
| 모바일 (< 600px) | 화면 하단 55% 높이 패널 |
| 태블릿 (≥ 600px) | 오른쪽 사이드 패널 (320px) |
| 데스크탑 (≥ 900px) | 오른쪽 사이드 패널 (380px) |

### 3. DSS 이미지 배경

설정 메뉴에서 **DSS 이미지 배경** 체크박스를 활성화하면 현재 조준 좌표의 실제 천체사진(STScI Digitized Sky Survey)을 별자리 지도 뒤에 오버레이합니다.

#### 동작 방식

- 3초마다 현재 중심 좌표 확인 → 0.3° 이상 이동 시 새 이미지 요청
- 이미지 로딩 중 서서히 나타나는 페이드인 효과
- 별자리 선·천체 기호는 이미지 위에 그대로 유지 (CSS `mix-blend-mode: lighten` 적용)
- 온라인 전용 — 오프라인 시 자동으로 숨김

#### 이미지 출처

[STScI Digitized Sky Survey (DSS)](https://archive.stsci.edu/cgi-bin/dss_search) — POSS-II/UKSTU Red 필드

## PWA 설치 (오프라인 사용)

모바일 브라우저 주소창 → **"홈 화면에 추가"** → 앱처럼 사용 가능

핵심 기능(별자리 지도, 관측 가이드, 자이로 방향 추적)은 오프라인에서도 동작합니다.
DSS 이미지 배경은 인터넷 연결이 필요합니다.
