# AstroHopper KR

망원경 스타 호핑을 위한 한국어 웹 앱 — [원본 AstroHopper](https://artyom-beilis.github.io/astrohopper.html) 기반 개선판

**▶ 바로 사용하기:** https://bih-lab.github.io/Astrohopper_BIH.ver/

---

## 추가된 기능

| 기능 | 설명 |
|------|------|
| 🇰🇷 한국어 UI | 모든 버튼, 메뉴, 안내 메시지 한국어화 |
| 🔭 관측 가이드 | 현재 조준 좌표 주변 천체 목록 (거리·등급·크기 표시) |
| 🌌 DSS 이미지 배경 | STScI DSS 실제 천체사진을 지도 배경에 오버레이 |

## 사용 방법

1. 스마트폰 브라우저에서 위 링크 접속
2. 위치 정보 및 기기 센서 권한 허용
3. 망원경에 스마트폰 고정 후 **정렬** 버튼으로 기준별 설정
4. **가이드 ▲** 버튼으로 주변 천체 목록 열람
5. 천체 탭 → 자동으로 지도로 복귀 + 방향선 표시

## PWA 설치 (오프라인 사용)

모바일 브라우저 주소창 → **"홈 화면에 추가"** → 앱처럼 사용 가능
