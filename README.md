# AstroHopper KR

망원경 스타 호핑을 위한 한국어 웹 앱 — [원본 AstroHopper](https://artyom-beilis.github.io/astrohopper.html) 기반 개선판

**▶ 바로 사용하기:** https://bih-lab.github.io/Astrohopper_BIH.ver/

---

## 기본 사용법

1. 스마트폰 브라우저에서 위 링크 접속
2. 위치 정보 및 기기 센서 권한 허용
3. 망원경에 스마트폰 고정 후 **정렬** 버튼으로 기준별 설정
4. **가이드 ▲** 버튼으로 주변 천체 목록 열람
5. 목록에서 천체 탭 → 자동으로 지도로 복귀 + 방향선 표시

---

## 추가된 기능

### 1. 한국어 UI

원본의 영어 인터페이스 전체를 한국어로 번역하였습니다.

- 버튼, 메뉴, 안내 메시지 모두 한국어
- 언어 선택 드롭다운에서 한국어/영어 전환 가능
- 시작 가이드 팝업 한국어화

---

### 2. 관측 가이드 패널

화면 우하단의 **가이드 ▲** 버튼을 탭하면 현재 망원경이 향하는 방향 주변의 천체 목록을 표시합니다. 스타 호핑 중 다음 목표 천체를 빠르게 찾을 수 있습니다.

#### 전체 흐름

```
[별자리 지도 화면]
┌─────────────────────────┐
│                         │
│      별자리 지도         │
│                         │
│                [가이드 ▲]│  ← 우하단 버튼 탭
└─────────────────────────┘
          │
          ▼  (패널 열림 + 배경 차단)

[가이드 패널 — 모바일: 화면 하단 절반]
┌─────────────────────────────────────┐
│ [← 지도로]    RA 5h32m  Dec +22.1°  │  ← 현재 조준 좌표
│ 반경 10°  ━━━●━━  등급 9.0  ━━━●━  │  ← 슬라이더 필터
│ [산개✓][구상✓][성운✓][은하✓][이중 ]  │  ← 종류 토글 (이중성 기본 OFF)
│ [거리순 ↕]                           │  ← 정렬 버튼
├──────────────────────────────────────┤
│ 천체      종류    등급   거리    🔭   │  ← 컬럼 헤더
│ ● M42     성운   ★4.0   5.8°   🔭   │
│ ● M43     성운   ★9.0   6.1°   🔭   │
│ ● NGC1981 산개   ★4.6   7.2°   🔭   │  ← 고도 정상 (밝은 글씨)
│ ● M78     성운   ★8.3  11.4°   🔭   │  ← 고도 < 20° (흐린 글씨)
└──────────────────────────────────────┘
```

#### DSS 미리보기 — 🔭 버튼

각 행 오른쪽의 🔭 버튼을 탭하면 해당 천체의 실제 천체사진을 인라인으로 펼쳐볼 수 있습니다.

```
[🔭 버튼 탭 전]
● M42     성운   ★4.0   5.8°   🔭

[🔭 버튼 탭 후 — 로딩 중]
● M42     성운   ★4.0   5.8°   🔭
  로딩 중...                          ← 검정 배경, 암적응 방해 없음

[이미지 로드 완료]
● M42     성운   ★4.0   5.8°   🔭
  ┌───────────────┐
  │               │
  │  DSS 실제     │  300 × 300 px
  │  천체사진     │  STScI DSS 제공
  │               │
  └───────────────┘
  M42 — DSS

[🔭 버튼 재탭 → 접힘]
● M42     성운   ★4.0   5.8°   🔭
```

- 이미지 출처: [STScI Digitized Sky Survey](https://archive.stsci.edu/cgi-bin/dss_search) (POSS-II Red)
- 이미지 크기: 천체 크기에 따라 10~30 arcmin 동적 계산
- 로딩 중 흰 화면 없음 — 검정 배경 유지로 야간 암적응 보호
- 한 번 로드한 이미지는 재탭 시 즉시 표시 (브라우저 캐시)
- 오프라인 또는 API 오류 시 "이미지 없음" 표시

#### 필터 및 정렬

| 항목 | 범위 / 옵션 | 기본값 |
|------|------------|--------|
| 반경 슬라이더 | 5° ~ 30° | 10° |
| 한계등급 슬라이더 | 4.0 ~ 14.0 mag | 9.0 |
| 천체 종류 토글 | 산개성단 / 구상성단 / 성운 / 은하 / 이중성 | 이중성 제외 활성 |
| 정렬 방식 | 거리순 / 밝기순 | 거리순 |

#### 목록 컬럼 설명

| 컬럼 | 내용 |
|------|------|
| ● | 천체 기호. **흐린 글씨**: 현재 고도 20° 미만 (관측 조건 나쁨) |
| 천체 | 이름 (M42, NGC1981 등) — 탭하면 해당 천체로 이동 |
| 종류 | 산개성단 / 구상성단 / 성운 / 은하 / 이중성 |
| 등급 | 겉보기 등급 (★ 기호, bold) |
| 거리 | 현재 조준 방향과의 각거리 (°) |
| 🔭 | DSS 실제 천체사진 미리보기 토글 |

#### 반응형 레이아웃

| 화면 크기 | 패널 위치 |
|----------|----------|
| 모바일 (< 600px) | 화면 하단 55% 높이 패널 |
| 태블릿 (≥ 600px) | 오른쪽 사이드 패널 (320px) |
| 데스크탑 (≥ 900px) | 오른쪽 사이드 패널 (380px) |

#### 천체 선택 동작

천체 이름(행)을 탭하면:
1. 가이드 패널 닫힘
2. 선택한 천체로 방향선 표시
3. 상단 검색창에 천체 이름 자동 입력

## PWA 설치 (오프라인 사용)

모바일 브라우저 주소창 → **"홈 화면에 추가"** → 앱처럼 사용 가능

별자리 지도, 관측 가이드, 자이로 방향 추적 모두 오프라인에서 동작합니다.
DSS 미리보기 이미지는 인터넷 연결이 필요합니다.
